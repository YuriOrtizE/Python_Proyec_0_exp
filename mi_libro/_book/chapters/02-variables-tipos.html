3.1 Módulo 1: Fundamentos de Python

3.2 Objetivo del módulo

Adquirir los conceptos básicos de programación en Python: variables, tipos de datos, entrada/salida por consola, operadores, estructuras condicionales y buenas prácticas iniciales.

3.3 Variables y tipado dinámico

Una variable es un nombre que hace referencia a un valor almacenado en memoria. En Python no es necesario declarar el tipo; se infiere al asignar un valor.

Mostrar código 
x = 10
y = 3.14
nombre = "PID"
activo = True

print(f"x={x}, tipo={type(x)}")
print(f"y={y}, tipo={type(y)}")
print(f"nombre={nombre}, tipo={type(nombre)}")
print(f"activo={activo}, tipo={type(activo)}")
x=10, tipo=<class 'int'>
y=3.14, tipo=<class 'float'>
nombre=PID, tipo=<class 'str'>
activo=True, tipo=<class 'bool'>

3.5 Operadores aritméticos

Mostrar código 
a, b = 10, 3
print(f"Suma: {a + b}")
print(f"Resta: {a - b}")
print(f"Multiplicación: {a * b}")
print(f"División: {a / b}")
print(f"División entera: {a // b}")
print(f"Módulo: {a % b}")
print(f"Potencia: {a ** b}")
Suma: 13
Resta: 7
Multiplicación: 30
División: 3.3333333333333335
División entera: 3
Módulo: 1
Potencia: 1000

3.6 Operadores de comparación

Mostrar código 
a, b = 10, 3
print(f"a > b: {a > b}")
print(f"a == b: {a == b}")
print(f"a != b: {a != b}")
a > b: True
a == b: False
a != b: True

3.7 Operadores lógicos

Mostrar código 
verdadero = True
falso = False
print(f"True and False: {verdadero and falso}")
print(f"True or False: {verdadero or falso}")
print(f"not True: {not verdadero}")
True and False: False
True or False: True
not True: False

3.8 Estructuras condicionales (if-elif-else)

Mostrar código 
temperatura = 85

if temperatura > 100:
    estado = "Sobrecalentamiento"
elif temperatura > 80:
    estado = "Alerta amarilla"
else:
    estado = "Normal"

print(f"Temperatura: {temperatura}°C - Estado: {estado}")
Temperatura: 85°C - Estado: Alerta amarilla

3.9 Ejercicios prácticos

3.9.1 Ejercicio 1: Calculadora de parámetros de motor

Mostrar código 
import math

voltaje = 220
corriente = 5
velocidad_rpm = 1500

potencia = voltaje * corriente
par = (potencia * 60) / (2 * math.pi * velocidad_rpm)

print(f"Potencia eléctrica: {potencia:.2f} W")
print(f"Par motor: {par:.3f} Nm")
Potencia eléctrica: 1100.00 W
Par motor: 7.003 Nm

3.9.2 Ejercicio 2: Conversor de temperaturas

Mostrar código 
def celsius_a_fahrenheit(c):
    return (c * 9/5) + 32

def fahrenheit_a_celsius(f):
    return (f - 32) * 5/9

print(f"25°C = {celsius_a_fahrenheit(25):.1f}°F")
print(f"77°F = {fahrenheit_a_celsius(77):.1f}°C")
25°C = 77.0°F
77°F = 25.0°C

3.9.3 Ejercicio 3: Validación de rango

Mostrar código 
presion = 75

if 0 <= presion <= 100:
